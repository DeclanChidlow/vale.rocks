Services

I run a range of services from my home server and various servers scattered around the interweb. Some of the more public services I host include:

Fediverse

 Status

Personal instance of Akkoma for accessing the fediverse. The instance itself is invite only, but I’m available to follow as @vale@fedi.vale.rocks. Hosted on my home server.

AutoMod

 Status

Official deployment of my full-scale moderation bot for Revolt servicing in excess of 4,000 servers and tens of thousands of users. Hosted on a Hetzner VPS.

Search

 Status

A super lightweight search engine wrapper that allows users to search across websites from a single interface with quality of life additions. Hosted on GitHub Pages.

Jellyfin

 Status

Free and open source media server on which I host my media library containing a wide assortment of films, books, audiobooks, and shows for my viewing pleasure. The instance is invite-only and not accepting sign-ups. Hosted on my home server.
